1. Representation and Visualization of Data
-------------------------------------------
Big question: how do you turn real-world data into data that a computer
can understand?

Outline of Objectives:
- Introduce the basic data representation of scikit-learn:
  a matrix of size n_samples x n_features
- Show how to use scikit-learn's dataset loaders.
- Note that features must be like-valued
  (example: iris photos vs iris measurements)
- Use digits to show how 2D image data can be transformed into 1D features
- Use a random projection to visualize the digits data

Exercise:
- Load & visualize the olivetti faces data.  Use random projections to
  scatter-plot the data.

Reference:
- Text data: http://scikit-learn.github.com/scikit-learn-tutorial/working_with_text_data.html
